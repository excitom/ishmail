Composition Window Menu Bar

The menu bar at the top of the Composition window contains a row of four menu titles. The options in the menu bar are specific to functions you can perform in the Composition window.

Options in the Composition window menu bar enable you to modify your messages and your composition environment.

When you click on a menu title, the menu "pulls down," revealing a list of menu items, or options, that enable you to perform various actions on mail messages you compose and on the environment of your Composition window. The Help pull-down menu lets you choose help for any window component or menu item in the Composition window.

The four pull down menus are:

  • Message -Lists operations you can perform on messages. See Composition Message Menu.

  • Edit - Lists operations you can perform on the format of your message text. See Composition Edit Menu.

  • Options - Customizes display and processing options. See Composition Options Menu.

  • Help - Displays help text. See Help Menu Options.
