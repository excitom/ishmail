Reading Window Reply Menu

This section describes the options available in the Reading window's Reply pull-down menu. To see the menu options, click the Reply button or type Meta-R. The following menu appears:

Figure: Reading Window Reply Pull-down Menu

Reading Window Reply Menu Options

The Reply pull-down menu provides options for replying to the current message, for forwarding it, and for resending it.

Reply Options

These options are the same as the Reply options in the Main window Message pull-down menu and are described in Main window Message menu Composition Options.

Forward/Resend Options

These options are the same as the Forward and Resend options in the Main window Message pull-down menu and are described in Main window Message menu Composition Options.
