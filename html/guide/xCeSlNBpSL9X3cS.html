Reading Window Message Menu

This section describes the options available in the Reading window's Message pull-down menu. To see the Message menu options, click the Message button or type Meta-M. The following menu appears:

Figure: Reading Window Message Pull-down Menu

Reading Window Message Menu Options

The Message pull-down menu provides options for reading, saving, and printing messages, filtering messages, marking messages for deletion, and closing the Reading window.

Read Options

These options are the same as the Read options in the Main window Message pull-down menu. They are described in Main window Message menu Read Options.

Save Options

These options are the same as the Save options in the Main window Message pull-down menu. They are described in Main window Message menu Save Options.

Print/Filter Options

These options are the same as the Print ... and Filter ... options in the Main window Message pull-down menu. They are described in Main window Message menu Printing and Filtering Options.

Delete/Close Options

Undelete
This option is only available if you are reading a message which has been marked for deletion. Click Undelete to restore this deleted message.

Mark for Deletion
Click this option to mark the message for deletion upon the next save changes operation or upon exiting the program with the Save changes option selected.

If you have specified the Hide messages marked for deletion option in the Mail Preferences window, all messages marked for deletion will be removed from the Folder Contents listing. You can restore (undelete) these messages before performing a save operation by choosing the Undelete from list... option in the main window Message menu.

(For more information about restoring messages that have been marked for deletion, see the Undelete last, Undelete selected, and Undelete from list options in the Main window Delete Options.)

Mark for Deletion/Close
Click this option to mark the current message for deletion and close the Reading window. The delete operation takes effect upon the next Save changes action or upon exiting the program with the Save changes option selected.

Close
Click Close to exit the current Reading window.
