Send Options

Select one of the send options to submit your composed messages to the sendmail subsystem. The three send options are:

Send/Keep Text
Click this option to send the message and keep the Composition window open. The message headers and message body remain intact.

Send
Click this option to send the message and keep the Composition window open. The message headers and message body are cleared.

Send/Close
Click this option to send the message and close the Composition window.
