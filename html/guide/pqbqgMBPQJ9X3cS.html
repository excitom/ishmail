Clear or Close the Composition Window

Select the Clear option to clear some or all of the fields in the Composition window. When you click the Clear option, a confirmation window is displayed that offers two clear-field options: Clear headers and Clear body.

These options are useful if you want to send a different message to the same recipients or send the same message to different recipients.

Select the Close option to abandon the current message and exit from the Composition window.
