Main Window Message Menu

This section describes the options available in the Message pull-down menu. To see the Message menu options, click the Message button, or type Meta-M. The following menu appears:

Figure: Message Pull-down Menu

Message Menu Options

The Message menu consists of functions that operate on individual messages or groups of messages selected from the active folder. Messages in the active folder are listed in the Folder Contents viewing area. (For more information about the active folder and its contents, see Open Folders Viewing Area and Folder Contents Viewing Area.)

The behavior of many of the options in the Message pull-down menu can be modified by selections in the Preferences ... option in Main Window Options Menu.

To choose an option in the Message pull-down menu, either click on the option, type its associated mnemonic (the underscored letter in the option's name), or type its associated Ctrl-key sequence.
