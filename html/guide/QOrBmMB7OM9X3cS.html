Exiting Ishmail

To exit from the Ishmail application, you can either:

  • Select the Exit option from the File pull-down menu (or type Ctrl-E while in the Main window).

  • Select the Quit option from the File pull-down menu (or type Ctrl-Q while in the Main window).

Exit Option

Select the Exit option if you wish to save changes you have made to mail folders.

When you select Exit, the Exit confirmation window appears.

Figure: Exit Confirmation Window

If you select No, you are returned to the application. If you select Yes, AND if you have made changes to mail folders that have not yet been saved, the Save confirmation window appears.

Figure: Save Confirmation Window

Select Yes to save changes and exit. Select No to discard changes and exit. Select Cancel to return to the application.

A Help button is also available if you need additional help on the exit options.

Quit Option

Select the Quit option if you wish to quit Ishmail without saving changes you have made to mail folders.

When you select Quit, the Exit confirmation window appears.

Figure: Exit Confirmation Window

If you select No, you are returned to the application. If you select Yes, the Ishmail application exits without saving any changes you have made to mail folders.
