Display Source

You can display the MIME-encoded representation of Rich Text mark-up attributes and of non-ASCII file-types, such as GIF, Binary, PostScript, and so forth by selecting the Display source option. The displayed source includes the MIME content-type headers and all MIME-encoding. The Display source function is useful if you wish to see the original format of a MIME-encoded message.

Click the Display source option to display the MIME-encoded source for ASCII Rich Text and for non-ASCII file-types. Ishmail launches a special text window to display the text.

Three examples of the Display source output are shown below. Examples one and two show (partial) source for included and attached GIF files. The third example shows the source for a Rich Text message.
