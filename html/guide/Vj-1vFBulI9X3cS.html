Composition Window

This section describes Ishmail's Composition window. You use the Composition window when you want to compose a new mail message, reply to a message, or add text to a message you are forwarding. This section covers:

  • Composition Window Components

  • Opening the Composition Window

  • Composing and Sending a Message

  • Composition Window Menu Bar

  • Composition Message Menu

  • Composition Edit Menu

  • Composition Options Menu

NOTE:
A modified version of the Composition window, called the Comment on Application window, can be used to send a comment to H&L Technical Support relating to the application. It is described in Comment on Application Window. Window components in the Comment on Application window share much of the functionality of the Composition window components described below.
