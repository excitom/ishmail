Help Menu

Ishmail offers a comprehensive Help facility that provides help messages for all window and menu components. Seven types of help information are available.

To get help while in the Composition window, click the Help pull-down menu, then click on the appropriate help subject. For detailed information about using the Help menu system, see Help Menu Options.
