Load Mail Template

Select the Load ... option to load the Composition window from a file containing a template for a mail message. Unlike the Include/Attach -> option, this does not insert the contents of the file into the message you are composing. Instead, it replaces the contents of the Composition window, if any, with the contents of the template file.
