Ishmail tar archive

If you install the tar archive, be aware of these limitations:

  • You (and other users) will need to use the full path name of the program, since it will not be installed in a common place, such as /usr/bin.
  • You may see a warning message that you cannot lock your InBox folder. If you see this, you will not be able to make changes to your InBox. This is because the Ishmail program needs "set group ID" privilege in the "mail" group, and setting this privilege requires root privilege on your system.
