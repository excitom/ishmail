Fix up tar installation

If you didn't install the tar archive with root privilege, these commands will fix problems with locking you InBox folder, and also make the program more easily accessible.

First, go to the directory where you installed Ishmail, then go to the bin subdirectory.

su
chgrp mail ishmail.real
chmod 2755 ishmail.real
ln -s `pwd`/ishmail /usr/bin/ishmail
