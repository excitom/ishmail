Using FTP to download files

Ishmail files can be downloaded using anonymous FTP, where the "FTP" stands for "File Transfer Protocol." The file transfer is "anonymous" since you do not need to have a login account on our system.

Note: If you use your Web browser program to download you do not need to know about the details of anonymous FTP. Follow the steps outlined here to download using your Web browser.

Proceed to the next step or, if you are interested in the details, here they are:

  • Use FTP to connect to ftp.ishmail.com
  • When prompted for login, type anonymous
  • When prompted for password, type your e-mail address
  • Type bin to set binary transfer mode
  • Type cd /pub/ishmail to go to the main Ishmail directory
  • Type cd current to go to the current version directory.

    Files are named like this:
    ishmail-version-date-system-type-package-compression

  • Type get file name to get the file you have chosen
  • Type quit to exit from FTP
