Ishmail has been built on a variety of Unix systems, including:

  • IBM AIX
  • Hewlett-Packard HP-UX
  • Digital Unix
  • SunOS
  • Sun Solaris (SPARC)
  • Sun Solaris (Intel x86)
  • Linux
  • SGI IRIX
  • BSDI
  • FreeBSD

To obtain a version of the program built for other than Solaris, HP-UX, or Linux you must download the source code and build it yourself.

Ishmail has a graphical user interface which requires the X Window System, version X11R4 or newer. Also, Ishmail uses OSF Motif graphical libraries, which are built into the program so your system does not need to have Motif run-time libraries installed.

Ishmail is packaged in two different formats, a tar archive and a package customized for the operating system's installation commands. Select here for more information on package types.
