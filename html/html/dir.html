Download Directory

This is the directory into which the Ishmail package will be downloaded. If you are installing the tar archive, this will also be where the package is installed.

If you are using the system-specific package, you will most likely install the package in a different directory. In fact, the installation program may select the directory for you and not give you a chance to change it.

This directory should have enough free space to contain the complete, uncompressed package.
