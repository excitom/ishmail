Support for Novell UnixWare

Ishmail is built and tested on UnixWare 2.0. It is automatically integrated with the UnixWare desktop.

Two types of packages are provided, a tar archive and a package which can be installed using the pkgadd command. Select here for more information on package types.

UnixWare support is being discontinued

Considering that Novell has abandoned UnixWare, and also considering lack of customer demand, Ishmail support for UnixWare is being discontinued.

The current version is available on an "as-is" basis. No further licenses will be sold.
