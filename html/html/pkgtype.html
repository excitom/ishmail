Ishmail package types

You may wish to install your demo copy of Ishmail as a private copy. Or, you may not have root privilege on your system, so you can't use the system's installation commands. Or, you want the quickest, easiest method of installation. If any of these apply to you, then install the tar archive version of Ishmail.

Note: There are some drawbacks to using the tar version.

If you want to use your operating system's "official" installation command, you want your demo copy to be available to other users, and you have root privilege on your system, then you should install the package version of Ishmail. The specific command you will use depends on the system type.
