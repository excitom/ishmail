Message List Option

The Message List ... option lets you change the column order and the column width of message status information in the window. You can also elect to show or to hide the message status icon.

Specifically, you control:

  • The titles shown for each column of information.

  • The order in which the columns are shown.

  • Which columns are shown or hidden.

  • The minimum width for each column.

  • The maximum width for each column.

  • Whether status icons are shown for each message.

  • The format used in the Date column.

NOTE:
You can also change the width of columns in the message list by clicking on the vertical separator bars above the message list, and dragging right or left. Changes made this way are only temporary.

To modify or reorder message status information, click the Message List ... option. The Message List Preferences window appears.

Figure: Message List Preferences Window

The Message List Preferences window contains a row of toggle buttons, each of which controls an associated message status Field (or status column). The fields are:

  • Number

  • Status

  • Sender

  • Subject

  • Date

  • Line count

  • Byte count

  • Show status icon

  • Date format

The default value for all message status fields is On. To remove any field from the Main window message status list, simply toggle the button associated with that field to Off. When you apply this change, the field is removed from the message list.

Adjacent to each toggle button is set of four text fields in which you can enter values to specify the Position, Minimum Width, Maximum Width, and Title of each message status field. These values are applicable to message status fields that are toggled On.

As you specify your preferences for each message status Field, you can apply your changes interactively at any time to see how they look.

Specifying Status Field Preferences

The procedure for specifying your message status field preferences is simple. For each message status Field (or status column) that is toggled On:

  1. Enter a value in the Position text field to designate the Field's relative position (from left-to-right) in the Main window message status list.

    The lowest number in a Position text field places its associated message Field furthest left in the message list; the highest number places its associated message Field furthest right.

  2. Enter a value in the Minimum Width text field to designate the minimum width (in characters) of the field.

    A non-zero value means the field will always display the specified number of characters, even if all of the entries are smaller.

  3. Enter a value in the Maximum Width text field to designate the maximum width (in characters) of the field.

    A value of zero or blank means the field will always expand to accommodate the longest line of text. A non-zero value means the field will never display more than the specified number of characters, even if one of the field's entries is longer.

  4. Enter a title in the Title field (or you can accept the default name, if one is available).

  5. Toggle the Show status icon either On or Off according to your preference.

Date Format

You can modify the format used to display the Date column by specifying the format using the syntax of the strftime(3) library function.

Each conversion specification is replaced by appropriate characters as described in the following list. The appropriate characters are determined by the LC_TIME category of the program's locale, as shown in Date Format.

Table: Date Format

The following example illustrates the use of the date format string. Assuming the date is Thursday, August 28, 1986 at 12:44:36, and a format string:

%A %b %d %j

This results in date containing Thursday Aug 28 240.

The default for the Date format field is a blank string, meaning that the date will take the format of the original message.

Applying Your Changes

Apply or cancel your changes using the row of buttons across the bottom of the window. See Applying Your Changes.
