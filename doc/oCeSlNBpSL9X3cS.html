Reading Window Menu Bar

The menu bar at the top of the Reading window contains a row of four menu titles. The options in the menu bar are specific to the Reading window.

When you click on a menu title, the menu "pulls down," revealing a list of menu items, or options, that enable you to perform various actions on mail messages in the Reading window or to modify the Reading window environment. The Help pull-down menu lets you choose help for any window component or menu item in the Reading window.

The four pull down menus are:

  • Message -Lists operations you can perform on a message in the Reading window. These options are described in Reading Window Message Menu.

  • Reply - Lists the reply and forward operations. These options are described in Reading Window Reply Menu.

  • Options - Customizes display and processing options. These options are described in Reading Window Options Menu.

  • Help - Displays help text. See Help Menu Options.
