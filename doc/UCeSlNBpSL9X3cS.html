Opening a New Reading Window

There are several ways to open a Reading window. From the Main window, you can:

  • Double-click a message in the Folder Contents area of the Main window.

  • Select one of the Read -> options from the Main window Message pull-down menu. (See Read Options.)

  • Place the mouse pointer on any message in the Folder Contents area and press the right mouse button to display a pop-up menu from which you can select the Read option. (See Message Pop-up Menu.)

Using an Existing Reading Window

Once you have opened a Reading window, you can leave it open and continue using it to display other messages by:

  • Selecting one of the Next -> or Previous -> options from the Reading window Message pull-down menu. This action removes the current message and displays a different message in the window. (See Reading Window Menu Bar.)

  • Using the "Read" functions in the Main Window. (See Main window Read Options and Message Pop-up Menu Options.

  • Dragging-and-dropping a message from the Folder Contents area of the Main window into an open Reading window.

    To do this, place the mouse pointer on the message and press-and-hold the center mouse button. When an icon for the message appears, drag the icon to the Reading window and release the middle mouse button. The existing message in the Reading window is replaced the new message.

NOTE:
Options are available in both the Main window Message pull-down menu and the Reading window Message pull-down menu for reading the next or previous unread message, the next or previous message from the same sender, the next or previous message with the same subject, and so forth. These are explained in Read Options in the Main window chapter and in Reading Window Message Menu Options in this chapter.
