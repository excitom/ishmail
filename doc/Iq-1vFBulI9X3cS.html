Composition Message Menu

This section describes the options available in the Composition window's Message pull-down menu. To see the Message menu options, click the Message button or type Meta-M. The following menu appears:

Figure: Composition Message Pull-down Menu

Composition Message Menu Options

The Message pull-down menu provides options for invoking an alternate text editor, for previewing messages, for sending messages you have created or modified, for including or attaching external files to those messages, and for clearing or closing a Composition window.
