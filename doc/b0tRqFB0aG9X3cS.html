Main Window Menu Bar

The menu bar at the top of the Main window contains a row of five menu titles. When you click on a menu title (or type the appropriate keyboard sequence), the menu "pulls down," revealing a list of menu items, or options. These options enable you to perform various actions on your mail folders and files. The Help pull-down menu lets you choose help for any window component or menu item.

The pull-down menus are:

  • File menu - Lists file options. See Main Window File Menu.

  • Folder menu - Lists operations on message folders. See Main Window Folder Menu.

  • Message menu - Lists operations on individual messages. See Main Window Message Menu.

  • Options menu - Customizes display and processing options. See Main Window Options Menu.

  • Help menu - Displays help text. See Help Menu Options.

Menu Characteristics

Each pull-down menu can be "torn off" by selecting the dotted line at the top of the pull-down menu. Tear-off menus can be moved around and placed anywhere on your screen display. They remain displayed until you explicitly close them. See the X User's Guide for more information about using pull-down and tear-off menus.

Some menu selection items are followed by an ellipsis ( . . . ). When you click on the item, a pop-up dialog window appears that prompts you for additional information or gives you additional selection options. The figure Pop-up Window shows an example of this type of window.

Using the Keyboard

Ishmail uses standard Motif key bindings as well as some custom mappings.

Menu items can also be selected with special keyboard shortcut sequences and mnemonic abbreviations.

Keyboard Shortcuts

Many commonly used functions have an associated shortcut key. The Control (Ctrl) key is used to activate shortcuts. If a function has a shortcut key, you will see its name to the right of the function name in a menu. For example, pull down the File menu. The Print and Exit functions have shortcut keys (Ctrl-P and Ctrl-E, respectively).

Mnemonic Abbreviations

A mnemonic abbreviation is indicated by the underscored letter in a menu item, such as the a in the Read menu item and the P in the Print menu item. To select the Read option, for example, you would simply type the character a when the Message pull-down menu is displayed.

NOTE:
It is recommended that you use a standard three-button mouse to traverse Ishmail's menu interface; however, special keyboard sequences are also available for making menu selections and for moving around in Ishmail windows. See Keyboard Conventions for more information on how to use key sequences.

On some keyboards, you can also start some Ishmail features using the Meta key in combination with a mnemonic abbreviation.

NOTE:
On Sun keyboards, the Meta key is the key with the diamond symbol, to the right of the Alt key. On systems that don't have a Meta key, the Alt key provides this function.
