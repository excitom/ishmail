Alternate Text Editor Option

Select the Editor ... option to launch an alternate text editor program for composing a mail message. When you perform the save/quit function in the alternate editor, the contents of that file appear in the Composition window.

See the Composition Window Preferences dialog for information about specifying an alternate editor of your choice.

NOTES:

  1. It is not possible to completely bypass the Composition window when using an alternate editor.

  2. If you have used functions in the Edit menu, and if your message type is MIME, you may see MIME Enriched Text markup tags mixed in with the message text. If you delete these tags you will undo their effect on the text. You can manually add tags if you like, though you will likely find doing so in the Composition window is much easier.
