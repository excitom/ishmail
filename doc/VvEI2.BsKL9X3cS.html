Check Addresses Now

Select the Check addresses now option to cause Ishmail to send all the addresses you have entered into the header fields to the sendmail program, which will validate whether they can be used to deliver mail.

NOTE:
If one or more addresses are outside of your local network, it may take a while for this operation to complete.
