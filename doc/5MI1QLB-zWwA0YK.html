Displayed Headers Option

Choose the Displayed headers ... option to specify the headers you want to be displayed in the headers display area of the Reading window. Headers not specified for display will be ignored in the headers display area.

To specify your header preferences, click the Displayed headers ... option. The Edit Displayed Headers window appears.

Figure: Edit Displayed Headers Window

Displayed Headers Preferences

Mail messages frequently contain numerous headers, most of which are of interest only to the underlying sendmail program. To make the header display box a bit more readable, you can filter out all but the headers that are of interest to you, such as From:, To:, Date:, Cc:, and Subject:.

The Edit Displayed Headers window is designed to let you either choose headers to display or choose headers to ignore.

Stated another way, you can specify a list of headers that you want to be displayed, indicating that Ishmail should ignore all other headers; or you can specify a list of headers that you want to be ignored, indicating that Ishmail should display all the rest.

Specifying Headers to Be Displayed or Ignored

The Edit Displayed Headers window is divided into two lists. On the left is a list of headers to be displayed. On the right is a list of headers to be ignored. You need only specify headers in one of the areas, then select that area to be either displayed or ignored, and apply your changes.

Between each list arrow buttons point in opposite directions. Pressing the appropriate arrow button causes one or more selected headers to be moved from one list to the other.

You can also drag-and-drop headers from one list to the other using the middle mouse button.

Moving Headers between Lists

To move the header comments from the list on the right (Use Ignore List) to the list on the left (Use Display List):

  1. Select the comments header. The left-pointing arrow between the two lists is highlighted, indicating it is now active.

  2. Once highlighted, you can either:

    • Drag-and-drop the header into the list on the left, or

    • Click the left-pointing arrow to move the header to the list on the left.

Removing or Clearing Headers

To remove or clear headers from either list:

  1. Select one or more headers in a list.

  2. Click the Remove or Clear button as appropriate.

    A pop-up confirmation box prompts you for confirmation before completing the operation.

Using the Text Field

To add, remove, or clear a header from either list using the text field below the list:

  1. Enter the name of the header in the text field.

  2. Click the Add, Remove, or Clear button as appropriate.

    A confirmation box prompts you for confirmation before completing the operation.

Selecting the List to Be Displayed or Ignored

Below the selection boxes are two radio buttons that determine if the values in the Use Display List or the values in the Use Ignore List should be used. Click the appropriate button.

Applying Your Changes

Apply (or cancel) your changes using the row of buttons across the bottom of the window. See Applying Your Changes.
