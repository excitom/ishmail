Opening the Composition Window

You can open the Composition window by several different methods. The method you choose depends on whether you want to compose a new message, reply to a message, resend a message, forward a message, or comment on the application. (To comment on the application, you use the modified Composition window called the Comment on Application window. (See Comment on Application.)

To open the Composition window to compose a new message:

  1. Select the Compose New ... option from the Message pull-down menu in the Main window menu bar; or

  2. Type Ctrl-N while in the Main window.

To open the Composition window to reply to the current or selected message:

  1. Select the Reply option from the Message pull-down menu in the Main window menu bar.

  2. From the cascading menu that appears, choose one of the four reply options:

    • To sender

    • To sender (include message)

    • To all

    • To all (include message)

      See Composition Options for an explanation of the reply options.

NOTE:
If you know the Ctrl-key sequence for the reply option you intend to use (see Cascading Menu), you can bypass the Message pull-down menu altogether and simply type the appropriate Ctrl-key sequence from the Main window.

To open the Composition window to reply to any message, even if it is not the selected message:

  1. Place the mouse pointer on the message and press the right mouse button.

  2. A pop-up menu appears (see Message Pop-up Menu) from which you can choose one of the four reply options.

    • To sender

    • To sender (include message)

    • To all

    • To all (include message)

    • See Composition Options for an explanation of the reply options.

To open the Composition window to resend the current or selected message:

  1. Select the Resend ... option from the Message pull-down menu in the Main window menu bar.

  2. Supply the appropriate headers and resend the message.

    See Composition Options for an explanation of the Resend ... option.

NOTE:
When resending a message, the Subject: and message body fields are grayed out, and you cannot change them. This is because when you resend a message it appears to the recipient as if it came from the original sender rather than from you.

To open the Composition window to resend any message, even if it is not the selected message:

  1. Place the mouse pointer on the message and press the right mouse button.

  2. A pop-up menu appears (see Message Pop-up Menu) from which you choose the Resend ... option.

    See Composition Options for an explanation of the Resend ... option.

To open the Composition window to forward the current or selected message:

  1. Select the Forward ... option from the Message pull-down menu in the Main window menu bar.

  2. Supply the appropriate header information and forward the message.

    See Composition Options for an explanation of the Forward ... option.

To open the Composition window to forward any message, even if it is not the selected message:

  1. Place the mouse pointer on the message and press the right mouse button.

  2. A pop-up menu appears (see Message Pop-up Menu) from which you choose the Forward ... option.

    See Composition Options for an explanation of the Forward ... option.

To open the Comment on Application window:

  1. Select the Comment ... option from the File pull-down menu in the mail window menu bar.

  2. Fill in the appropriate comment and header information and send the message.

    See Comment on Application Window.
